# Agent Skills

Reusable skills for AI coding agents. Install with the [skills](https://skills.sh) CLI.

[![skills.sh](https://skills.sh/b/Gusen1453/agent-skills)](https://skills.sh/Gusen1453/agent-skills)

## Available skills

### designing-test-cases

How to design unit/integration test cases, assertions, and mocks — without coverage theater or formalistic tests that restate the implementation.

**Install:**

```bash
npx skills add Gusen1453/agent-skills --skill designing-test-cases
```

Or install everything in this repo:

```bash
npx skills add Gusen1453/agent-skills
```
